# The neuropsychology of object constancy (Turnbull, Carey & McCarthy, JINS 1997)

## TL;DR

**Object constancy** = recognising an object across viewpoints. Turnbull et al. argue the
**primary** route to recognition is **viewpoint-invariant** and lives in the **ventral
(occipitotemporal) stream**; a **viewpoint-dependent** route (mental rotation / viewer-
centred coding), supported by **occipitoparietal** structures, is a **back-up** used only
under non-optimal conditions. Patient dissociations map onto this two-stream split.

## Topic context

The theoretical backbone for **object orientation agnosia**: it explains how a patient can
recognise an object yet not know its orientation (intact ventral viewpoint-invariant route,
damaged dorsal/parietal viewpoint-dependent route).

## Method / argument structure

Critical review tying cognitive theories of object recognition to the **two cortical visual
systems** account (Ungerleider & Mishkin; Milner & Goodale) using neuropsychological,
animal single-cell, and lesion evidence.

## Key findings

- **Two theory families:** *viewpoint-dependent* (mentally rotate the image to a canonical
  view, then match — viewer-centred; Jolicoeur, Tarr & Pinker) vs *viewpoint-invariant*
  (object-centred description about the principal axis — Marr; or geons — Biederman).
- **Two streams:** ventral (occipito-temporal) = object recognition with **object-centred /
  viewpoint-invariant** codes; dorsal (occipito-parietal) = spatial/visuomotor with
  **viewer-centred** codes (Milner & Goodale). So viewpoint-invariant recognition ≈ ventral.
- **Dissociations supporting the split:**
  - **Unusual-views deficit** (right inferior parietal lesions): recognise canonical views,
    fail foreshortened/unusual views — a *parietal* contribution to recognition under hard
    conditions (the IPL may *bind* viewer- and object-centred info).
  - **Mental rotation can dissociate from recognition** either way: occipitotemporal lesions
    → lost recognition but **spared** mental rotation (LH); parietal lesions → impaired
    mental rotation but **spared** recognition (RT).
  - **Agnosia for object orientation / spontaneous rotation** (LG, NL, SC; Turnbull et al.
    1995/1997): name objects but cannot give their canonical upright orientation; copy
    drawings rotated 90/180°. → access to viewpoint-invariant identity, loss of
    viewer-centred orientation (parietal/temporo-parietal lesions).
  - **Mirror-image discrimination** deficits: cannot tell an object from its mirror image —
    information **not available** in a viewpoint-invariant code; temporal-lobe lesions in
    monkeys *improve* mirror discrimination (they treat mirror images as different).
- **Conclusion:** viewpoint-invariant recognition (ventral) is primary; viewpoint-dependent
  processing (dorsal/parietal) is an **optional resource** for non-optimal images.
